# Lab2_Node.js

## Ініціалізація проєкту
npm init -y

## Встановлення бібліотек
npm install axios

## Перевірка версії бібліотеки
npm list axios

## Запуск програми
node app.js

## Встановлення залежностей з package.json
npm install

## Перевірка вразливостей
npm audit

# npm audit виведе список CVE для пакетів, якщо вони є
# приклад: axios 1.7.5 — CVE-2024-XXXX

## Оновлення пакетів (за потреби)
npm update
