# Lab4_Python

## Запуск

## Створення віртуального оточення
python -m venv venv

Windows: venv\Scripts\activate

## Встановлення залежностей
pip install -r requirements.txt

## Оновлення пакету (за потреби)
pip install --upgrade pip

## Запуск програми
python main.py
